# E-Commerce Microservices

Cloud-native e-commerce backend built with microservices architecture.

## Services
- **User Service** — Authentication & user management
- **Product Service** — Product catalog & inventory
- **Order Service** — Order management

## Tech Stack
- Node.js + TypeScript
- Docker + Kubernetes
- Helm
- GitHub Actions CI/CD
- Prometheus + Grafana

## Getting Started
```bash
docker-compose up
```
